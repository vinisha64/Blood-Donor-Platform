# 🩸 Blood Donor Platform

A full-stack MERN application that connects blood donors with hospitals through a secure, location-based platform. It enables donor registration, hospital management, blood request handling, and geolocation-based donor search with role-based access control.


---

## ✨ Features

- Secure JWT Authentication
- Role-Based Access Control (Admin, Donor, Hospital)
- Donor & Hospital Registration
- Admin Approval System
- Blood Request Management
- Geolocation-Based Donor Search
- Interactive Donor Map
- Real-Time Notifications
- Donation Eligibility Tracking
- Responsive User Interface

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/vinisha64/Blood-Donor-Platform.git
```

Navigate to the project

```bash
cd Blood-Donor-Platform
```

Install dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

Start the application

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```
